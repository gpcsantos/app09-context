# App 09 - react context

Projeto tem a finalidade de estudar useContext

## Branch main se destina aos alunos, com um projeto minino de exemplo.

## Funcionalidades

- vite
- react
- react router dom
- tailwindcss
- json server

### Pasta Data

- db.json

# Para rodar o projeto

- yarn - instala dependencias
- yarn start - inicia o Vite (porta padrão) e o JSON server (porta 3000)

### REUSMO

O React Context é uma funcionalidade que permite compartilhar dados globalmente entre vários componentes, sem precisar passá-los manualmente através de props em cada nível (prop-drilling). Ele é ideal para armazenar estados como temas (claro/escuro), dados de usuário autenticado ou preferências de idioma
